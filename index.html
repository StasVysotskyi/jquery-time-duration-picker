jquery-time-duration-picker

jQuery-ui widget for choosing time duration

View on GitHub Download .zip Download .tar.gz

Demo

Installation

bower install jquery-time-duration-picker

Using

<script>
  $(function() {
    $('#duration').timeDurationPicker({
      defaultValue: function() {
        return 12345;
      },
      onselect: function(element, seconds, duration) {
        $('#seconds').val(seconds);
        $('#duration').val(duration);
      }
    });
  });
</script>
Also, see this example.

Options

Name Type Default Description
seconds boolean false Display or not seconds
minutes boolean true Display or not minutes
hours boolean true Display or not hours
days boolean true Display or not days
months boolean true Display or not months
years boolean true Display or not years
defaultValue function | integer   Initial value
onselect function(element, seconds, duration)   Callback function
